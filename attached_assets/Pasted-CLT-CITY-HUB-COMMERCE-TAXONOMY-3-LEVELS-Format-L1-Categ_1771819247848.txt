CLT CITY HUB — COMMERCE TAXONOMY (3 LEVELS)
Format:
L1 = Category (Top-Level)
  L2 = Subcategory (Second-Level)
    L3 = Micro Categories (Service/Focus Tags — searchable, used heavily in Enhanced)

GLOBAL RULES (Architect)
- L1/L2 are stable (rare changes). L3 is flexible (search tags).
- Verified: 1 L2 + 0–3 L3 (or 0 L3 if you prefer)
- Charter: 1 L2 + up to 3 L3
- Enhanced: up to 3 L2 + up to 12 L3 (hard cap)
- Voting/Reviews attach to a Commerce Entity AND optionally to a specific L2 (so “Best Roofer” works).
- Awards/Top-10 are NOT categories; they are campaigns that reference L1/L2 + neighborhood + year.
