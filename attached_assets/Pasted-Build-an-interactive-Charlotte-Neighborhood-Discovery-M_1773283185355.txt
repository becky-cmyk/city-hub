Build an interactive Charlotte Neighborhood Discovery Map for the relocation section of the City Hub platform.

The purpose of this map is to help people moving to Charlotte explore neighborhoods and discover the local hubs connected to each area.

This is NOT a traditional real estate map. It is a community discovery map that connects users to neighborhood hubs, businesses, events, and Pulse stories.

PAGE LOCATION

/charlotte/moving-to-charlotte

Add a section titled:

Explore Charlotte Neighborhoods

--------------------------------------------------

FEATURE 1 — Interactive Map

Create an interactive Charlotte metro map showing major neighborhoods and surrounding communities.

Each map point should represent a neighborhood hub.

Examples:

Uptown  
South End  
NoDa  
Plaza Midwood  
Dilworth  
Ballantyne  
SouthPark  
University Area  
Matthews  
Huntersville  
Cornelius  
Pineville

Belmont

When a user clicks a neighborhood:

Open a preview panel with:

Neighborhood name  
Short description  
Lifestyle tags  
Featured local businesses  
Upcoming events  
Link to the neighborhood hub page

Example link:

/hub/south-end
