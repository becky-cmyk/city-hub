PART 1 — CREATE PLATFORM PRICING TABLE
--------------------------------------------------

Create new table: platform_products

Fields:
- id (uuid, pk)
- name (string)
- category (enum: listing, hub, addon, capability, crown, contributor, promo)
- billing_type (enum: one_time, monthly, annual)
- stripe_product_id (string, nullable)
- stripe_price_id (string, nullable)
- price_version (string, default "v1")
- active (boolean, default true)
- created_at
- updated_at

PART 6 — CREATE METRO LAUNCH WORKFLOW (CORE)
--------------------------------------------------

Create service:
openMetro({ name, slug })

Steps:
1. create metro record
2. create metro_launch_project
3. call createComingSoonPage
4. set metro status = "coming_soon"
5. generate checklist:

checklist_json example:
{
  "setup": ["define regions", "seed hubs"],
  "content": ["create starter articles"],
  "outreach": ["build business list"],
  "monetization": ["identify sponsors"]
}

Return:
{
  metro,
  launch_project,
  coming_soon
}
