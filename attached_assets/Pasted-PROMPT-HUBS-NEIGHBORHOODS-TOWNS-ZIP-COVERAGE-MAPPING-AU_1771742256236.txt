1) DATABASE — HUB ↔ ZIP MAPPING
------------------------------------------------------------

1A) Create table hub_zip_coverage
- id (uuid)
- hub_region_id (FK -> regions.id, CASCADE)
- zip (FK -> zip_geos.zip, CASCADE)
- confidence enum: "high" | "med" | "low" default "low"
- notes text nullable
- created_at
Unique index: (hub_region_id, zip)

1B) Create table hub_geo_snapshot (optional helper)
- hub_region_id (FK -> regions.id, PK)
- center_lat numeric nullable
- center_lng numeric nullable
- created_at
(This avoids changing regions schema if you don’t want new columns.)
