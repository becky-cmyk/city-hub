- Resolve city by citySlug (must exist).
- Determine Stripe priceId from productType (+ tier if listing tier).
- Create/reuse stripe customer:

  - Email source:
    - If subjectType=BUSINESS: owner_email must exist; else reject

- Create Stripe Checkout Session:
  - Mode:
    - LISTING_TIER can be subscription OR one-time (pick ONE and implement; default subscription if possible)
    - All others default one-time payment (mode="payment")

    city_id, city_slug, subject_type, subject_id, product_type, tier (if any)
