Prompt 1 — Expand the existing CLT City Hub / City Hub platform without creating a separate system. Build everything off the current Verified Hub Presence architecture and existing microsite/profile model. Do not reinvent the wheel. Extend the current business listing / hub presence so it can support additional badges, modules, content types, and opportunity systems under the same base profile.

CORE RULES
1. Everything must remain under the same hub brand and same hub presence system.
2. Do not build separate directories for podcasts, speakers, contributors, creators, or experts.
3. Use the existing hub presence / microsite as the base entity.
4. Add modular badges, profile sections, permissions, and upgrade logic on top of the existing profile.
5. Keep the architecture scalable for micro hubs, metro content, business content, and future media/network features.
6. Preserve the existing micro hub / ZIP code / category logic.
7. All paid participation must require a Verified Hub Presence.
8. The free/open community submission layer can still exist for general story/event submission, but advanced tools, visibility, and network participation must tie back to account level and paid status.

BUILD / EXPAND THE PLATFORM AROUND THIS MODEL

A. VERIFIED HUB PRESENCE FOUNDATION
Use Verified Hub Presence as the base paid membership and profile system.

Verified Hub Presence:
- Base price: $299
- Includes:
  - one primary ZIP code / micro hub
  - one primary category
  - one primary microsite / profile
  - eligibility for paid upgrades
  - account-based content management
  - eligibility for media network participation
  - eligibility for business/expert/speaker/creator/podcast modules

Geographic upsells:
- additional ZIP code or micro hub: $50 each
- full Charlotte metro coverage upgrade: $2,000

Location logic:
- ZIP code is the territory anchor
- if business has a physical location, that location ZIP is the primary ZIP
- if mobile/service business, billing ZIP is the primary ZIP
- additional physical locations beyond the first: $99 each

Category logic:
- one primary category included
- additional categories are an upsell
- micro-categories should also be supported as a deeper discovery/search upsell
